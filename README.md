### 2024-10-13
- 주제 : Kotlin의 `when`과 `setOf()` 이해  
- 내용 요약:  
  - `when`은 Java의 `switch`와 비슷한 구문으로, 객체와 집합도 인자로 사용할 수 있다.  
  - `setOf()`는 중복을 허용하지 않으며, 순서에 상관없이 집합을 비교할 수 있다.  
  - `when` 내부에서 `setOf()`가 여러 번 호출되면 불필요한 객체가 생성될 수 있으므로 최적화가 필요하다.
